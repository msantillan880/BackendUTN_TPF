3) PRUEBAS BACKEND CON POSTMAN
====================================================================
Objetivo: validar endpoints y reglas de autorizacion con evidencia tecnica.

Configurar variables de coleccion:
- baseUrl = https://backendutn-tpf.onrender.com
- ownerEmail = prueba.usuario2.bookmarksutn@gmail.com
- ownerPassword = bookmarksutn.2026
- visitorEmail = prueba.usuario1.bookmarksutn@gmail.com
- visitorPassword = bookmarksutn.2026

B1. Swagger/API docs
- GET {{baseUrl}}/api-docs
- Esperado: docs accesible.

B2. Login owner
- POST {{baseUrl}}/api/auth/login
- Body:
  {"email":"{{ownerEmail}}","password":"{{ownerPassword}}"}
- Esperado: 200 + accessToken (guardar en accessToken).

B3. Auth me owner
- GET {{baseUrl}}/api/auth/me
- Header: Authorization Bearer {{accessToken}}
- Esperado: 200 + user.

B4. Owner crea espacio privado
- POST {{baseUrl}}/api/espacios
- Header auth bearer
- Body:
  {"denominacion":"EspacioPostmanDemo","tipoEspacio":"privado"}
- Esperado: 201 + idEspacio (guardar espacioId).

B5. Login visitante
- POST {{baseUrl}}/api/auth/login
- Body:
  {"email":"{{visitorEmail}}","password":"{{visitorPassword}}"}
- Esperado: 200 + accessToken visitante.

B6. Visitante solicita ingreso al espacio
- POST {{baseUrl}}/api/espacios/{{espacioId}}/solicitudes
- Esperado: 201 (pendiente).

B7. Visitante intenta crear link sin aprobar
- POST {{baseUrl}}/api/links
- Body:
  {"idEspacio":{{espacioId}},"nombre":"Prueba sin permiso","comentario":"Debe fallar","direccion":"https://example.com"}
- Esperado: 403.

B8. Owner aprueba solicitud
- (loguear nuevamente owner y usar su token)
- PUT {{baseUrl}}/api/espacios/{{espacioId}}/solicitudes/{{visitorUserId}}/aprobar
- Esperado: 200.

B9. Visitante crea link luego de aprobacion
- (token visitante)
- POST {{baseUrl}}/api/links
- Body:
  {"idEspacio":{{espacioId}},"nombre":"Prueba autorizada","comentario":"Luego de aprobacion","direccion":"https://www.wikipedia.org"}
- Esperado: 201.

B10. Export HTML y log
- POST {{baseUrl}}/api/espacios/{{espacioId}}/generar-html
- GET {{baseUrl}}/api/log-view?token={{accessToken}}
- Esperado: export correcto y logs visibles.
